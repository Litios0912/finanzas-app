Aplicación de finanzas personales con React Native y Supabase.

## ✨ Características

- 📊 Dashboard con gráficos de gastos
- 💳 Control de gastos fijos y dinámicos
- 🏦 Gestión de ahorros
- 👨‍👩‍👧‍👧 Soporte multi-familia
- 🔐 Autenticación con Supabase
- 📱 Diseño responsive

## 🚀 Instalación

```bash
npm install
npx expo start
```
